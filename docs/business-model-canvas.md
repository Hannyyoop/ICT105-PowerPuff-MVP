# Business Model Canvas

## Project Title

Smart Study Room Booking System

## 1. Customer Segments
University students who need study rooms for individual study or group discussions.

## 2. Value Proposition

The system helps students quickly find and reserve available study rooms online. It reduces the time spent searching for empty rooms, prevents double bookings, and makes room management easier for library staff.

## 3. Channels

- Desktop and mobile web browser

## 4. Customer Relationships

- Easy-to-use booking interface
- Booking confirmation and status updates
- User-friendly navigation
- Staff support when needed
## 5. Revenue Streams / Sustainability Logic

This is not a commercial product.

The system creates value by:
- Reducing staff workload
- Improving the use of study rooms
- Saving students' time
- Supporting better study experiences on campus

## 6. Key Resources

- HTML, CSS, and JavaScript
- Internet connection
- Study room information
- Student booking data
- Staff

## 7. Key Activities

- Display available study rooms
- Process room bookings
- Update room availability
- Manage booking records
- Allow staff to manage reservations

## 8. Key Partners

- University 
- Staff
- Students who use the system
- Faculty members (for feedback and testing)

## 9. Cost Structure

- Website development time
- Design using Figma or Canva
- Web hosting
- System maintenance
- Staff training
- Internet and server resources

## Final Note

This business model supports the MVP by focusing on the essential features: viewing available study rooms, booking a room, checking booking status, and managing bookings. These features solve the main problems identified during user research and provide a simple, efficient prototype for students and staff.
